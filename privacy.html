Privacy Policy

Effective Date: 12/08/2025

Overview

This Chrome extension (“Extension”) is designed to help users identify whether links in Google Search results have already been pitched in Pitchbox, and to display relevant project information.

We take privacy seriously. This Extension does not collect, store, or share any personal information from users, except as required for its core functionality.

Information We Collect

  • Non-personal data: The Extension reads URLs from Google Search result pages to check them against the Pitchbox database.
  • Cached data: Pitchbox results may be stored locally in your browser’s storage to improve performance.
  • API requests: The Extension communicates with the Pitchbox API to retrieve pitch status data. These requests are authenticated through your own Pitchbox account.

How We Use Information

  • To display pitch status badges on search results.
  • To show the related Pitchbox project if a link has been pitched.
  • To improve performance by caching results locally.

Data Sharing

We do not sell, rent, or trade your data.

Data is only sent to Pitchbox’s API to retrieve pitch information. No other third parties are involved.

Data Storage and Retention

  • All cached results and settings are stored locally in your browser and can be cleared at any time.
  • No personal browsing history is collected or stored.

Permissions Justification

The Extension requests permissions (storage, unlimitedStorage, activeTab, tabs, alarms, and appropriate host_permissions) solely to enable its single purpose: checking Pitchbox status for SERP links and displaying badges.
